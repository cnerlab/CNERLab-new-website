32-Channel EEG Acquisition System

High-resolution EEG system for brain-computer interface and neural signal acquisition. Supports real-time data streaming and advanced signal processing.

  • 32 active channels with high input impedance
  • Sampling rate: up to 1 kHz per channel
  • 16-bit/24-bit ADC resolution
  • USB and wireless connectivity
  • Real-time data export (LabVIEW, MATLAB, Python APIs)
  • Integrated trigger input/output
  • Compatible with standard EEG caps

    Arduino Mega 2560 Kit

    Microcontroller kit for rapid prototyping and hardware interfacing. Includes sensors, shields, and accessories for embedded system projects.

    • ATmega2560 microcontroller (16 MHz)
    • 54 digital I/O pins (15 PWM outputs)
    • 16 analog inputs
    • 256 KB flash memory, 8 KB SRAM
    • USB interface for programming and power
    • Includes assorted sensors and shields
    • Compatible with Arduino IDE
